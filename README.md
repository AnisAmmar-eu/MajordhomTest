## Stack
- **Frontend** : React 18
- **Backend** : Node.js + Express
- **Base de données** : MongoDB (Mongoose)

## Lancer le projet

### 1. Backend
```bash
cd backend
npm install
node server.js
```

### 2. Frontend
```bash
cd frontend
npm install
npm start
```
